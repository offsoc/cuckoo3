# Creating virtual machines

## All-in-one
```bash
vmcloak --debug init --win10x64 --hddsize 128 --cpus 2 --ramsize 4096 --network 192.168.30.0/24 --vm qemu --vrde --vrde-port 1 --ip 192.168.30.2 --iso-mount /mnt/win10x64 win10base br0
vmcloak --debug install win10base --recommended
vmcloak --debug snapshot --count 3 win10base win10vm_ 192.168.30.10
```

## Network

The analysis machine must have their own network and default gateway. The 
default gateway is where Cuckoo3 will start the network capture for traffic from 
and to a specific machine.  
VMCloak will handle assigning unique IPs to each snapshot.

!!! tip "Network guidelines and advice"

    - For the best analysis results, analysis machine should be able to reach the 
    internet.
    - Analysis VMs should not be able to reach other host ports other than the 
    result server one.

## Creating an image 

!!! note "Requirements"

    Before you create your virtual machines, make sure that:

    - VMCloak installed, please see [Installing VMCloak](../installing/vmcloak.md){:target=_blank}.
    - You have downloaded preferred image, please see [Downloading an image](../installing/vmcloak.md#downloading-an-image){:target=_blank}.
    - You created a network interface and mount, please see [Configuring VMCloak](../installing/vmcloak.md#configuring-vmcloak){:target=_blank}.

An image is the disk where we install the chosen OS on. Any dependencies we choose are also installed on this. 
When we have made all the modifications we want, we make our virtual machines from this image. 
After this, the image must not be edited.

1. Create the image.

        vmcloak --debug init --win10x64 --hddsize 128 --cpus 2 --ramsize 4096 --network 192.168.30.0/24 --vm qemu --vrde --vrde-port 1 --ip 192.168.30.2 --iso-mount /mnt/win10x64 win10base br0

2. Install software on the image (see [Recommended installation](#recommended-installation)).

        vmcloak --debug install win10base --recommended

3. (Optional) Install custom software.  
(check [Usage / dependencies](#usage) for full list of installable software)

        vmcloak --debug install win10base office office.version=2010 office.isopath=/home/cuckoo/office2010.iso office.serialkey=XXXXX-XXXXX-XXXXX-XXXXX-XXXXX office.activate=1

4. To make snapshots, run:

        vmcloak --debug snapshot --count 3 win10base win10vm_ 192.168.30.10

Snapshots will be stored in `~/.vmcloak/vms/qemu/` directory.

## Recommended installation

VMCloak `--recommended` option in `vmcloak install` does the following:

- **Installs**:
    - ie11
    - .NET
    - Java
    - VC redistributables 2013-2019
    - Edge
    - Adobe PDF, 
- **Modifies**:
    - updates (Letâ€™s encrypt) root certs
    - a wallpaper
    - OS optimization (stopping updates, removing unneeded components)
    - disable unneeded services such as Cortana

This can all be viewed afterwards with the list image command.

## Adding machines to Cuckoo

```bash
cuckoo machine import qemu /home/cuckoo/.vmcloak/vms/qemu
cuckoo machine delete qemu example1
```
